# generator-nicknames
Это простой генератор никнеймов, написанный на Python. Он позволяет создавать никнеймы двух типов: осмысленные и случайные. Пользователь выбирает режим работы, затем вводит длину никнейма. В осмысленном режиме также можно выбрать тему: фэнтези, киберпанк или обычные слова. Программа случайным образом берёт слова из выбранной темы и соединяет их в один ник, пока не достигнет нужной длины. В случайном режиме ник создаётся из случайных букв английского алфавита до нужной длины. В программе используются модули random и string. Проект сделан в учебных целях для практики работы с условиями, циклами, строками и случайными значениями в Python.
